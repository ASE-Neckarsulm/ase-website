# ASE Website

This is the official website for ASE (Automatisierung Schaltanlagen Elektrotechnik), built with React, TypeScript, and Vite.

## Project Overview

The website showcases ASE's services, including:
- Switchgear Construction (Schaltschrankbau)
- Electrical Engineering/Planning (Elektrokonstruktion)
- PLC Programming (SPS-Programmierung)
- Robot Programming (Roboterprogrammierung)

## Tech Stack

- **Framework:** React
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** CSS
- **Deployment:** Netlify

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

Build the application for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Project Structure

- `/src` - Source code
  - `/components` - Reusable UI components
  - `/pages` - Application pages
  - `/assets` - Static assets (images, etc.)
  - `/data` - Static data files
